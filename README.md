# CanteenFoodOrdering Web Application Project
## Web Programming CourseProject in acdemic 2nd Year
develope by using Vue.js Express.js and bulmaCSS with MySQL database  

## Project setup
```bash
npm install 
```

## Project start
```bash
npm run serve
```
